## 🌐 3. API 연동 테스트

### 환경 설정 확인
- [ ] 개발 환경 (localhost) URL 확인
  - [ ] Backend: http://localhost:3001
  - [ ] Content Backend: http://localhost:8000
- [ ] 프로덕션 환경 URL 확인
  - [ ] Backend: https://artify-backend-3y4r.onrender.com
  - [ ] Content Backend: https://artify-content-api.onrender.com

### 세그먼트 API
- [ ] GET /segments - 세그먼트 목록 로드
- [ ] POST /segments - 세그먼트 생성
- [ ] PUT /segments/:id - 세그먼트 수정
- [ ] DELETE /segments/:id - 세그먼트 삭제

### AI 생성 API
- [ ] POST /generate/text - 텍스트 생성 (GPT-3.5)
- [ ] POST /generate/text - 텍스트 생성 (Gemini Pro)
- [ ] POST /generate/image - 이미지 생성 (DALL-E 3)
- [ ] POST /generate/image - 이미지 생성 (Stable Diffusion XL)
- [ ] GET /models - 사용 가능한 모델 목록

### 분석 API
- [ ] GET /analytics/summary?days=7 - 7일 데이터
- [ ] GET /analytics/summary?days=30 - 30일 데이터
- [ ] GET /analytics/summary?days=90 - 90일 데이터
