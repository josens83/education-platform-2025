# Artify Platform - 추가 기능 구현 계획

## 🎯 목표

기존 3개 페이지의 기능을 보완하고, 사용자 경험을 개선합니다.

---

## 📝 구현할 추가 기능

### 1️⃣ 세그먼트 → 생성 연동 강화 ⭐ 우선순위 높음

**현재 상태**:
- segments.html에서 "콘텐츠 생성" 버튼 클릭 시 segment_id를 URL 파라미터로 전달
- generate.html에서 segment_id를 받아 드롭다운에 선택

**개선 사항**:
- ✅ 세그먼트 정보를 프롬프트에 자동 주입
- ✅ 세그먼트별 추천 키워드 제안
- ✅ 세그먼트 정보 카드 표시

**구현 파일**:
- `frontend/js/generate.js` - 세그먼트 정보 활용 로직 추가

---

### 2️⃣ 생성 결과 저장 기능 ⭐ 우선순위 높음

**현재 상태**:
- 생성된 결과가 메모리에만 저장됨 (페이지 새로고침 시 사라짐)

**개선 사항**:
- ✅ localStorage에 생성 결과 저장
- ✅ "저장된 결과" 탭 추가
- ✅ 저장된 결과 불러오기/삭제 기능

**구현 파일**:
- `frontend/js/generate.js` - 로컬 저장 로직 추가
- `frontend/generate.html` - UI 업데이트

---

### 3️⃣ 생성 → 에디터 데이터 연동 강화 ⭐ 우선순위 높음

**현재 상태**:
- sessionStorage로 텍스트와 이미지 전달

**개선 사항**:
- ✅ 에디터에서 자동으로 캔버스에 배치
- ✅ 텍스트 객체 + 이미지 객체 자동 생성
- ✅ "AI 생성에서 가져옴" 표시

**구현 파일**:
- `frontend/js/editor.js` - sessionStorage 데이터 처리 로직
- (에디터 파일이 매우 크므로 신중하게 수정)

---

### 4️⃣ 분석 페이지 실시간 새로고침 ⭐ 우선순위 중간

**현재 상태**:
- 페이지 로드 시 한 번만 데이터 로드

**개선 사항**:
- ✅ "새로고침" 버튼 추가
- ✅ 자동 새로고침 옵션 (30초, 1분, 5분)

**구현 파일**:
- `frontend/analytics.html` - UI 추가
- `frontend/js/analytics.js` - 새로고침 로직

---

### 5️⃣ 검색 기능 강화 ⭐ 우선순위 중간

**현재 상태**:
- segments.html에만 검색 기능

**개선 사항**:
- ✅ generate.html에서 생성 결과 검색
- ✅ analytics.html에서 콘텐츠 검색
- ✅ 검색 하이라이트 기능

**구현 파일**:
- `frontend/js/generate.js` - 검색 로직
- `frontend/js/analytics.js` - 검색 로직

---

### 6️⃣ 에러 처리 개선 ⭐ 우선순위 중간

## 🚀 구현 우선순위

### Phase 1: 핵심 기능 (지금 구현) ⭐⭐⭐
1. **세그먼트 → 생성 연동 강화** - 사용자 경험 직접 개선
2. **생성 결과 저장 기능** - 데이터 유실 방지
3. **생성 → 에디터 연동 강화** - 워크플로우 완성

### Phase 2: 편의성 개선 (다음 구현) ⭐⭐
4. **분석 페이지 새로고침** - 실시간성 개선
5. **검색 기능 강화** - 대량 데이터 관리
6. **에러 처리 개선** - 안정성 향상

### Phase 3: UX 향상 (선택적 구현) ⭐
7. **로딩 상태 개선** - 체감 성능 향상
8. **모바일 UX 개선** - 모바일 사용성
