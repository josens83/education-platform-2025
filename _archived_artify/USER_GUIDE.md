## 개요

Artify Platform은 **"세그먼트 정의 → AI 생성 → 디자인 → 분석"**의 완전한 마케팅 콘텐츠 제작 워크플로우를 제공합니다.

### 핵심 가치 제안

- ✅ **타겟 맞춤 콘텐츠**: 세그먼트별로 최적화된 메시지 생성
- ✅ **멀티 AI 모델**: 4개 AI 모델 중 선택 (비용/품질 트레이드오프)
- ✅ **즉시 사용 가능**: AI가 생성한 콘텐츠를 바로 에디터에서 편집
- ✅ **데이터 기반 개선**: 성과 데이터를 통한 지속적 최적화

## 2️⃣ 세그먼트 관리

### URL
```
https://artify-ruddy.vercel.app/segments.html
```

### 목적
타겟 오디언스를 정의하여 **맞춤형 콘텐츠**를 생성합니다.

### 주요 기능

#### 세그먼트 생성

1. **"+ 새 세그먼트" 버튼 클릭**

2. **정보 입력**:
   ```
   이름: 20대 여성 직장인
   설명: MZ세대 여성을 위한 패션 아이템 타겟

   타겟 기준:
   - 연령대: 20대
   - 성별: 여성
   - 관심사: 패션, 뷰티, 라이프스타일
   - 지역: 서울, 경기
   ```

3. **"저장" 클릭**

#### 세그먼트 카드

생성된 세그먼트는 카드 형태로 표시됩니다:

```
┌──────────────────────────────────────┐
│ 20대 여성 직장인                      │
│ MZ세대 여성을 위한 패션 아이템 타겟    │
│                                      │
│ [20대] [여성] [패션, 뷰티] [서울]     │
│                                      │
│ 생성된 콘텐츠: 0    마지막 사용: 없음  │
│                                      │
│ [✨ 콘텐츠 생성] [수정] [삭제]        │
└──────────────────────────────────────┘
```

#### "✨ 콘텐츠 생성" 버튼

- **동작**: generate.html로 이동하면서 `segment_id` 전달
- **효과**:
  1. 세그먼트 자동 선택
  2. 키워드 자동 제안
  3. 프롬프트에 타겟 정보 주입

#### 검색 기능

- 세그먼트명으로 실시간 검색
- 다수의 세그먼트 관리 시 유용

## 3️⃣ AI 콘텐츠 생성

### URL
```
https://artify-ruddy.vercel.app/generate.html
```

### 목적
AI를 활용하여 **텍스트 + 이미지**를 동시에 생성합니다.

### 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  왼쪽: 옵션 패널       │  오른쪽: 결과 패널         │
│  (400px)              │  (나머지 공간)            │
└─────────────────────────────────────────────────────┘
```

### 왼쪽 옵션 패널

#### 1. 세그먼트 선택

```
🎯 타겟 세그먼트
[▼ 20대 여성 직장인]  (자동 선택됨)
```

**효과**:
- 선택 시 키워드 자동 제안
- 프롬프트에 타겟 정보 자동 주입

#### 2. 텍스트 생성 옵션

```
✍️ 텍스트 생성

AI 모델: [▼ GPT-3.5 Turbo ($$$)]
        [  Gemini Pro ($$)]

프롬프트 *: [봄 신상품 출시 홍보 문구]

톤앤매너: [▼ 친근한]
키워드: [20대, 여성, 서울, 패션]  (자동 제안됨)
최대 길이: [▼ 중간 (~500 토큰)]
```

**AI 모델 선택 가이드**:

| 모델 | 비용 | 품질 | 추천 용도 |
|------|------|------|-----------|
| GPT-3.5 Turbo | $$$ | ⭐⭐⭐⭐ | 일반적인 마케팅 카피 |
| Gemini Pro | $$ | ⭐⭐⭐ | 비용 절감이 필요할 때 |

#### 3. 이미지 생성 옵션

```
🎨 이미지 생성

AI 모델: [▼ DALL-E 3 ($$$$)]
        [  Stable Diffusion XL ($$$)]

이미지 프롬프트: [현대적인 패션 스타일의 여성 모델]

이미지 크기: [▼ 정사각형 (1024x1024)]
            [  세로형 (1024x1792)]
            [  가로형 (1792x1024)]

☑ 텍스트와 이미지 함께 생성
```

**AI 모델 선택 가이드**:

| 모델 | 비용 | 품질 | 특징 |
|------|------|------|------|
| DALL-E 3 | $$$$ | ⭐⭐⭐⭐⭐ | 최고 품질, 프롬프트 이해도 높음 |
| Stable Diffusion XL | $$$ | ⭐⭐⭐⭐ | 비용 절감, 빠른 생성 |

#### 4. 생성하기 버튼

```
[✨ 생성하기]
```

**클릭 시**:
1. 세그먼트 정보를 프롬프트에 주입
2. AI API 호출 (30초 ~ 1분)
3. 결과를 오른쪽 패널에 표시
4. **자동으로 localStorage에 저장**

### 오른쪽 결과 패널

#### 생성 결과 카드

```
┌────────────────────────────────────────────────┐
│ 생성 결과            GPT-3.5 + DALL-E 3        │
│ 🎯 20대 여성 직장인                            │
├────────────────────────────────────────────────┤
│                                                │
│ [텍스트]                     [이미지]          │
│ 이번 봄, 당신을 위한 완벽한   [패션 모델 사진]  │
│ 스타일이 준비되었습니다!                        │
│ MZ세대 여성분들을 위해...                      │
│                                                │
├────────────────────────────────────────────────┤
│ 💬 577 토큰  💰 $0.0023  🕒 방금 전  ✓ 저장됨 │
├────────────────────────────────────────────────┤
│ [🎨 에디터에서 열기] [🔄 재생성] [📋 복사]      │
│ [🗑️ 삭제]                                      │
└────────────────────────────────────────────────┘
```

**카드 정보**:
- 세그먼트명 표시
- 사용된 AI 모델
- 텍스트 + 이미지 함께 표시
- 토큰 사용량, 비용, 시간
- 저장 상태 표시

**액션 버튼**:

1. **🎨 에디터에서 열기**
   - `sessionStorage`에 데이터 저장
   - `editor.html?from=generate`로 이동
   - 에디터에서 자동으로 캔버스에 배치

2. **🔄 재생성**
   - 동일한 옵션으로 새로 생성
   - 다른 결과를 원할 때 사용

3. **📋 복사**
   - 텍스트를 클립보드에 복사
   - 다른 곳에 붙여넣기 가능

4. **🗑️ 삭제**
   - 저장된 결과에서 삭제
   - `localStorage`에서 제거

### 세그먼트 정보 자동 주입 원리

**사용자 입력 프롬프트**:
```
봄 신상품 출시 홍보 문구
```

**AI에게 전달되는 실제 프롬프트**:
```
[타겟 오디언스 정보]
타겟 세그먼트: 20대 여성 직장인
설명: MZ세대 여성을 위한 패션 아이템 타겟
연령대: 20대
성별: 여성
관심사: 패션, 뷰티, 라이프스타일
지역: 서울, 경기

위 타겟 오디언스에 맞춘 콘텐츠를 작성해주세요.

봄 신상품 출시 홍보 문구
```

**효과**:
- AI가 타겟에 맞는 언어 사용 (예: "여성분들", "MZ세대")
- 세그먼트 특성 반영 (예: 서울 기반 → "강남 핫플")
- 톤앤매너 자동 조정

### 저장 기능 (localStorage)

**자동 저장**:
- 생성 시 자동으로 `localStorage`에 저장
- 브라우저 새로고침해도 유지
- 최대 5MB까지 저장 가능

**저장 데이터 구조**:
```javascript
{
  id: 1699123456789,
  text: {
    content: "이번 봄, 당신을 위한...",
    model: "gpt-3.5-turbo",
    usage: { prompt_tokens: 45, completion_tokens: 532, ... }
  },
  image: {
    url: "https://...",
    model: "dall-e-3",
    size: "1024x1024"
  },
  segment: {
    id: 1,
    name: "20대 여성 직장인"
  },
  timestamp: "2025-11-08T04:30:00Z",
  saved: true
}
```

**확인 방법**:
- 브라우저 개발자 도구 → Application → Local Storage
- 키: `artify_saved_results`

## 4️⃣ 에디터

### URL
```
https://artify-ruddy.vercel.app/editor.html

```

### 목적

### AI 생성 콘텐츠 자동 배치

**generate.html에서 "에디터에서 열기" 클릭 시**:

1. **sessionStorage에 데이터 저장**:
   ```javascript
   {
     text: "이번 봄, 당신을 위한...",
     image: "https://..."
   }
   ```

2. **에디터에서 자동 감지**:
   - URL 파라미터 `from=generate` 확인
   - sessionStorage 데이터 로드

3. **캔버스에 자동 배치**:
   - 텍스트 객체 생성 (상단)
   - 이미지 객체 생성 (하단)
   - 레이아웃 자동 조정

### 주요 편집 기능

**텍스트 편집**:
- 폰트 변경
- 크기 조정
- 색상 변경
- 정렬 (좌/중/우)

**이미지 편집**:
- 크기 조정
- 위치 이동
- 회전
- 자르기

**도형 추가**:
- 사각형, 원, 삼각형
- 배경으로 활용

**레이어 관리**:
- 순서 변경
- 앞으로/뒤로 보내기

**내보내기**:
- PNG (고해상도)
- JPG (압축)
- PDF (A4)

## 5️⃣ 분석 대시보드

### URL
```
https://artify-ruddy.vercel.app/analytics.html
```

### 목적
캠페인 성과를 분석하고 **데이터 기반 인사이트**를 얻습니다.

### 주요 섹션

#### 1. KPI 카드 (4개)

```
┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│ 생성된 콘텐츠 │ │ 총 생성 비용 │ │ 캐시 적중률  │ │ 평균 응답시간 │
│     247      │ │   $15.43     │ │     42%      │ │    2.1s      │
│ ↑ +12%       │ │ ↓ -8% 절감   │ │ ↑ +5% 향상   │ │ ↓ -15% 개선  │
└──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘
```

#### 2. 차트 (3개)

**📈 생성 추이 (라인 차트)**:
- 최근 7일간 생성 건수
- 추세 파악

**🎯 모델별 사용 (도넛 차트)**:
- GPT-3.5: 45%
- Gemini Pro: 30%
- DALL-E 3: 15%
- Stable Diffusion XL: 10%

**💰 세그먼트별 비용 (바 차트)**:
- 20대 여성: $5.20
- 30대 남성: $4.10
- 전체: $3.80
- 40대 이상: $2.30

#### 3. 콘텐츠 성과 테이블

**상위 성과 탭**:
```
┌──────────────┬──────────┬──────────┬──────┬────────┬──────┐
│ 콘텐츠        │ 세그먼트  │ 모델     │ 날짜 │ 비용   │ 성과 │
├──────────────┼──────────┼──────────┼──────┼────────┼──────┤
│ 신제품 런칭   │ 20대여성  │ GPT-3.5  │ 11/7 │ $0.002 │ 우수 │
│ 할인 이미지   │ 전체     │ DALL-E 3 │ 11/6 │ $0.040 │ 우수 │
│ 브랜드 스토리 │ 30대남성  │ Gemini   │ 11/5 │ $0.001 │ 양호 │
└──────────────┴──────────┴──────────┴──────┴────────┴──────┘
```

**개선 필요 탭**:
- 성과가 낮은 콘텐츠
- 개선 방향 파악

## 고급 기능

### 1. 프롬프트 캐싱 (백엔드)

**동작 원리**:
- 동일하거나 유사한 프롬프트 감지
- 이전 결과 재사용
- API 호출 비용 절감 (30-50%)

**사용자 입장**:
- 투명하게 작동 (자동)
- 비용 절감 효과만 확인 가능

**확인 방법**:
- 분석 페이지에서 "캐시 적중률" 확인

### 2. 세그먼트별 키워드 제안

**동작 원리**:
1. 세그먼트 선택
2. criteria 데이터 파싱
3. 키워드 입력란에 자동 입력

**예시**:
```
세그먼트: 30대 남성 직장인
criteria: { age_range: "30s", gender: "male", interests: "골프, 와인", location: "강남" }

자동 제안 키워드: "30대, 남성, 골프, 와인, 강남"
```

### 3. 비용 추적

**gen_jobs 테이블**:
- 모든 AI 생성 작업 로깅
- 토큰 사용량 기록
- 실시간 비용 계산

**비용 계산 공식**:

**텍스트**:
```
cost = (prompt_tokens / 1000 * input_price) +
       (completion_tokens / 1000 * output_price)

GPT-3.5 Turbo:
- Input: $0.0005 / 1K tokens
- Output: $0.0015 / 1K tokens

Gemini Pro:
- Input: $0.00025 / 1K tokens
- Output: $0.0005 / 1K tokens
```

**이미지**:
```
DALL-E 3:
- 1024x1024: $0.040
- 1024x1792: $0.080
- 1792x1024: $0.080

Stable Diffusion XL:
- 1024x1024: $0.030
- 기타 크기: $0.030
```

### 4. 자동 저장 (에디터)

**동작 원리**:
- 5초마다 변경 감지
- 변경사항 있으면 PostgreSQL에 저장
- 실시간 "저장 중..." 표시

**사용자 혜택**:
- 작업 손실 방지
- 명시적 저장 불필요

## 팁과 요령

### 💡 Tip 1: 비용 최적화

**저렴한 모델부터 시도**:
1. Gemini Pro로 초안 생성 (저렴)
2. 만족스러우면 사용
3. 불만족 시 GPT-3.5로 재생성

**예상 절감액**: 40-50%

### 💡 Tip 2: 효과적인 프롬프트 작성

**구체적으로 작성**:
```
❌ 나쁜 예: "광고 문구"
✅ 좋은 예: "20대 여성을 위한 봄 신상품 출시 이벤트 홍보 문구, 친근한 톤, 이모지 포함"
```

**세그먼트 활용**:
- 세그먼트에 타겟 정보 상세히 입력
- 프롬프트는 간단히 (AI가 자동으로 확장)

### 💡 Tip 3: 재생성 전략

**첫 번째 결과가 마음에 안 들 때**:

1. **톤앤매너 변경**:
   - 전문적 → 친근한
   - 공식적 → 위트있는

2. **키워드 추가**:
   - 구체적 특징 추가
   - 감정 키워드 ("신나는", "편안한")

3. **모델 변경**:
   - GPT-3.5 ↔ Gemini Pro
   - DALL-E 3 ↔ Stable Diffusion XL

### 💡 Tip 4: 에디터 활용

**AI 생성 후 반드시 편집**:
- 텍스트 강조 부분 굵게
- 색상으로 브랜드 일관성
- 배경 추가로 가독성 향상

**템플릿 활용**:
- 소셜 미디어 (1:1)
- 배너 (16:9)
- 포스터 (A4)

### 💡 Tip 5: 분석 활용

**주기적 확인 (주 1회)**:
- 어떤 세그먼트가 효과적인가?

- 캐시 적중률은 향상되고 있는가?

**AI 인사이트 활용**:
- "새 인사이트 생성" 버튼 클릭
- 데이터 기반 전략 조정

### 💡 Tip 6: 워크플로우 자동화

**효율적인 루틴**:

## 자주 묻는 질문 (FAQ)

### Q1. 생성 결과가 저장되는 기간은?

**A**: `localStorage`에 저장되므로 **브라우저가 삭제할 때까지** 영구 보관됩니다. 단, 브라우저 캐시를 지우면 삭제됩니다.

**권장**: 중요한 결과는 "에디터에서 열기" → PNG/PDF로 내보내기

### Q2. 여러 세그먼트를 동시에 타겟팅할 수 있나요?

**A**: 현재는 한 번에 하나의 세그먼트만 선택 가능합니다. 여러 세그먼트를 타겟팅하려면:
1. 각 세그먼트별로 생성
2. 또는 "전체" 세그먼트 선택

### Q3. AI 생성 비용은 누가 부담하나요?

**A**: 현재는 플랫폼에서 부담합니다. 향후 사용자 쿼터 시스템이 도입될 예정입니다.

### Q4. 생성된 이미지의 저작권은?

**A**:
- **DALL-E 3**: OpenAI 정책에 따라 상업적 사용 가능
- **Stable Diffusion XL**: Stability AI 라이선스 확인 필요

**권장**: 각 모델의 이용약관을 확인하세요.

### Q5. 프롬프트 캐싱은 어떻게 작동하나요?

**A**:
- **해시 기반**: 동일 프롬프트 감지 (정확히 일치)
- **시맨틱 기반**: 95% 유사도 이상 감지 (의미 일치)

**예시**:
```
프롬프트 1: "봄 신상품 광고"
프롬프트 2: "봄 시즌 신제품 홍보"
→ 95% 유사 → 캐시 히트 → 비용 절감
```

### Q6. 에디터에서 생성된 콘텐츠가 자동 배치 안 돼요.

**A**:
1. URL에 `?from=generate` 파라미터 확인
2. 브라우저 콘솔 (F12) → Application → Session Storage 확인
3. `generatedContent` 키 존재 여부 확인

**해결**: generate.html에서 "에디터에서 열기" 버튼 사용

### Q7. 분석 데이터가 실시간으로 업데이트 안 돼요.

**A**:
- 페이지 로드 시 한 번만 데이터 로드됨
- 새로고침 (F5) 또는 날짜 범위 변경 시 업데이트

**향후 개선**: 자동 새로고침 기능 추가 예정
